# 仿制B站-前端代码（持续更新中）

# 部署过程

## 需要的环境

- node.js：推荐20.x版本
- npm：推荐10.x版本

## 具体部署过程

仓库提供前端工程源代码。推荐克隆main分支的源代码，master分支为开发测试。

- 选取一个文件目录，打开命令行界面，输入：`git clone -b main https://github.com/All1217/AniLink-Vue3.git`
- `cd AniLink-Vue3`进入项目根目录
- 输入`npm install`安装依赖
- 安装完成后在根目录输入`npm run serve`，当命令行显示网址时打开浏览器，在地址栏输入`http://localhost:8080/`
